Prefer the command line?

Every operating system ships a SHA-256 hasher. Run one of these and compare the output:

  1. macOS / Linux: shasum -a 256 document.pdf or sha256sum document.pdf
  2. Windows: certutil -hashfile document.pdf SHA256

A matching hash proves integrity (your copy is identical to the sealed copy), not origin. Trace a document to its official government source to establish where it came from.
